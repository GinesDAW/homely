# README


En la carpeta del proyecto puedes ejecutar los siguientes comandos:


---


## npm start


Inicia la aplicación en modo desarrollo.


Abre http://localhost:3000 en el navegador para verla.


La página se recargará automáticamente cuando guardes cambios.  
Si hay errores, aparecerán en la consola.


---



## npm run build


Genera la versión optimizada para producción en la carpeta `build`.


- Optimiza el rendimiento.
- Minifica los archivos.
- Añade hashes a los nombres para evitar problemas de caché.


Después de esto, la aplicación está lista para desplegar.
